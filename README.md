## COMPSCI 683 — AI Poker Agent

Two-player Limit Texas Hold'em agent using CFR+ (Counterfactual Regret Minimization).
The agent trains offline via self-play and plays at runtime using a pre-computed strategy lookup table.

### How the agent works

1. **Offline (train.py):** Two CFR+ agents play self-play games via PyPokerEngine.
   Each decision updates a regret table keyed by abstract info sets.
   After convergence the average strategy is saved to `strategy.pkl`.

2. **Online (raise_player.py):** `declare_action` maps the current game state
   to an abstract info set key, looks up the pre-trained probabilities, and
   samples an action. No computation happens at runtime beyond the lookup.

**Info set key:** `(street, hand_bucket, pot_bucket, raise_count, position)`
- `street`      — 0=preflop, 1=flop, 2=turn, 3=river
- `hand_bucket` — 0–7 by Chen formula (preflop) or hand type (post-flop)
- `pot_bucket`  — 0–4 by pot size in chips
- `raise_count` — 0–4 raises this street
- `position`    — 0=small blind, 1=big blind

Total keyspace: **1,600 entries** — small enough to train quickly, large enough
to cover all strategically distinct situations.

```bash
python train.py
```

Increase `N_ITERATIONS` in `train.py` for a stronger strategy.

### Game rules (Limit Texas Hold'em)

- 2 players, initial stack $10,000, small blind $10 / big blind $20
- Four streets: preflop → flop (3 cards) → turn (1 card) → river (1 card)
- Actions: `fold` | `call` | `raise` (fixed +$10 above opponent's bet)
- Maximum 4 raises per street
- Exceed the time limit → engine auto-folds your hand

For full rules see `https://umamherst.instructure.com/courses/34216/files/16269266/download?download_frd=1`.
